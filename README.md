Mein Ziel ist es, ein Unternehmen im Bereich **Zero-Knowledge Proofs (ZKP)** zu gründen. 
* **Konzept:** Eine kostenlose App für Nutzer, die sicheres Anmelden auf Webseiten ermöglicht, ohne dass der Anbieter Passwörter oder persönliche Daten speichern muss.
